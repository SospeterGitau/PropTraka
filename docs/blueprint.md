# **App Name**: PropTraka

## Core Features:
- **Mobile-First Dashboard**: Uber-style navigation and touch-optimized layout for managing properties on the go. Displays key metrics: Occuapncy, Revenue, Arrears, and Net Profit.
- **Global Accessibility**: Built-in multi-language support (i18n) catering to a diverse range of international users (13+ locales).
- **AI-Powered Insights**: Leveraging Genkit to analyze portfolio health, predict maintenance needs, and score portfolio diversity. Generates actionable insights to improve ROI.
- **Property Management**: Comprehensive CRUD operations for properties, including image management, mortgage tracking, and detailed unit specifications.
- **Revenue & Expense Tracking**: Ledger-based financial tracking with support for recurring payments, one-off expenses, and detailed categorization. Supports automated payments via M-Pesa, Airtel Money, and Pesapal.
- **Tenant Portal**: Dedicated views for tenants to view lease details and payment history (roadmap).
- **Smart Alerts**: Proactive notifications for lease expiries, overdue payments, and maintenance anomalies.

## Style Guidelines:
- **Design System**: Shadcn UI (using Radix Primitives and Tailwind CSS) for a consistent, accessible, and professional look.
- **Typography**: Inter (via Google Fonts) for clean readability across all device sizes.
- **Color Palette**: 
    - Primary: Slate-950 (Dark/Professional)
    - Accents: Violet/Purple (AI features), Green (Positive financial metrics), Red (Alerts/Arrears).
- **Layout**: 
    - **Desktop**: Sidebar navigation with responsive grid layouts.
    - **Mobile**: Bottom navigation bar (FAB for primary actions) with optimized touch targets (min 44px).
- **Interactions**: Subtle animations using Framer Motion or Tailwind classes for transitions, ensuring the app feels "alive" and responsive.
